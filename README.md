# animal-shelter
대덕인재개발원 중간프로젝트 PL박성우


 너와 나의 연결고리
유기동물 보호소 운영 프로그램

 프로젝트 개요
유기동물 보호소의 입양·봉사·후원·고객지원 업무를 통합 관리하는 웹 시스템

개발 기간: 2026.03.20 ~ 2026.04.07 (3주)
팀 구성: 5인 팀 / PL 담당
본인 담당: 관리자 페이지 전체, 입양·봉사·회원·게시판 시스템

기술 스택
Java 17 Jakarta EE Servlet MyBatis Oracle 21c Bootstrap 5.3 jQuery JSP
주요 기능

담당 로직
회원 가입/로그인 (BCrypt 암호화, 세션/Filter 권한 관리)
입양 신청 (Leaflet.js 지도, FullCalendar 예약)
봉사 모집/신청 (중복 방지, 선착순 마감)
5종 통합 게시판 (댓글·대댓글·추천·신고)
관리자 대시보드 전체
